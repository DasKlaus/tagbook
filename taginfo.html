TAGBOOK DEMO - TAG INFO

Murder

A work contains a murder when a character gets premeditatedly, unlawfully and purposefully killed by another.
Tag as important if this killing plays a major role for the characters or the plot.

Disambiguation

Consider whether or not the following tags might be a better fit:
sacrifice, assassination or terrorism if the murder serves a greater good.
manslaughter if the killing is not premeditated.
capital punishment if the killing is legal punishment for a crime.
suicide if the killer is also the victim.
murder attempt if the killing is not successful.
murder (mention) if the murder is not described or discussed in detail.
fatal accident if the killing was not purposeful.
murdochite if the book is actually about a mineral.

Synonyms

Works with the following tags will be included in search results for "murder" and vice versa:
homicide killing

Umbrella tags

Works tagged with "murder" will be included in searches for the following tags:
death crime violence felony
